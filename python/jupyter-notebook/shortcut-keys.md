# Shortcut Keys

> jupyter notebook의 단축키 정리



```bash
H : 단축키 목록 출력
A : 상단 셀 생성
B : 하단 셀 생성
클릭 or Enter : Edit Mode로 전환
esc : Command Mode로 전환
가위 아이콘 or X : 셀 삭제
Z : 셀 삭제 취소
ctrl + Enter : 현재 셀 실행
shift + Enter : 실행 + 다음 셀 선택(없다면 생성)
alt + Enter : 실행 + 다음 셀 생성
M : 마크다운 셀로 변경
Y : 코드 셀로 변경
ctrl + s or S : 저장
함수클릭 + shift + tab : Docstring
ctrl + shift + - : 셀 나누기
```
